基于 LightGBM + LLM 的可解释互动风控

从情报与人工审核发现的作弊账号出发，把零散经验转化为可规模化筛查、可解释复核、可沉淀复用的互动风控策略。

问题发现
  • 情报 / 人工审核发现一批作弊账号，但缺乏可泛化的识别与治理策略。
  • 特征依赖人工经验，难以规模化。
  • 阈值难以确定，策略不稳定。
  • 审核成本高，决策效率低。
风险定义
  • 风险类型：虚假关注、评论水军、营销笔记等。
  • 风险分级：根据规模、影响时间、影响程度划分低 / 中 / 高风险。
识别策略：模型 LightGBM
  • 通过风险类型观察总结，用粗策略去白，将大盘量级降低到千万，重点区分中高风险。
  • 拼接尽可能多类型的互动统计特征，用分类模型 LightGBM 对账号进行风险打分。
  • 用 TreeSHAP 计算特征边际贡献，推测作弊手法。
  • 调用 LLM 生成自然语言解释，提升策略可理解性与复用性。
  • 每日例行产出可以连接展示在前端看板。

风险分级处置、策略审核与持续迭代

处置策略

基于风险分数进行分级处置，在清除风险与控制误伤之间保持平衡。

  • 低风险：不处理 / 仅展示给自己（控制误伤）。
  • 中风险：限流 / 功能限制（降低影响）。
  • 高风险：账号封禁（清除风险）。
  • 策略审核：基于 SHAP 的异常打分自然语言解释，支持人工审核快速复查与手法理解，可制定相关策略。
  • 策略沉淀：将高风险特征组合转化为规则，迭代优化长期治理策略。
评估与迭代
  • 风控模型效果：多个 case 中识别精度 P80+，大部分接近 P100。
  • 支持千万级账号筛查，可识别百万级问题账号。
  • 治理效果：显著降低风险暴露，降低人工审核成本。
  • 提升识别效率，最快 2h 内可产出高质量结果。
  • 持续迭代：根据误判 / 漏判调整阈值更便捷。
  • 可解释的风险模式可以沉淀为通用策略与规则。

基于多视图深度学习的成本网络瓦解策略
2024.09 – 2025.01
  • 出发点：网络瓦解旨在给出可以快速破坏网络结构的删除节点序列。现有方法主要考虑无成本场景或仅考虑单节点删除成本，忽略了实际问题中的转移成本与转移规则。
  • 主要方案：提出两类新型成本网络模型——回合成本优化（攻击者每步可任意选择节点但需支付转移成本）与即时成本优化（攻击者每步需从邻居节点中选择下一个攻击点），并基于强化学习设计顺序攻击算法。
基于残差社区感知的通用图异常检测
2025.12 – 2026.04
  • 出发点：通用图异常检测旨在无需目标图标注与微调的情况下识别未见图异常节点。现有方法依赖监督信号或目标图适配，且难以应对异常语义迁移和模式多样的真实场景。
  • 主要方案：通过跨图特征对齐、高阶 ego-neighbor 残差建模和软社区分配，学习可迁移的异常结构模式；推理阶段结合社区原型偏差与邻域上下文不一致计算异常分数。
证据驱动的视觉语言模型安全评测框架
2026.03 – 2026.05
  • 出发点：视觉语言模型安全评测旨在判断模型能否基于图文证据做出正确安全决策。现有方法主要关注模型是否拒答或顺从，难以区分视觉感知、OCR、跨模态绑定、意图理解和安全策略执行中的具体失败原因。
  • 主要方案：提出 EviSafe 证据驱动的视觉语言模型安全评测框架。通过自然回复、证据报告和反事实变化三阶段探测，诊断模型是否基于正确多模态证据做出安全决策，并揭示现有模型行为稳定性上的不足。
